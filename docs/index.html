常见问题

Dailymatters 是免费的吗？

是的，Dailymatters 可在 App Store 免费下载使用。

可以设置重复事件吗？

可以。支持每日、每周、每月、每年等多种重复模式，非常适合生日、账单和日常打卡。

支持哪些提醒方式？

每个事件都可添加多个自定义提醒，支持当天提醒和提前提醒，确保重要时刻不错过。

我的数据安全吗？

所有事件数据都保存在设备本地，无需账号。使用 AI 助手时需要你自行配置 API 密钥，对话通过加密连接发送，我们不会存储你的密钥或对话记录。

有桌面小组件吗？

有。你可以把 Dailymatters 小组件添加到主屏幕，随时查看今日安排与重要倒计时。

支持哪些语言和系统版本？

支持简体中文与英文双语，需要 iOS 17.0 或更高版本。
